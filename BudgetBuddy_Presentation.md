# BudgetBuddy Presentation

## Slide 1: Project Title
- BudgetBuddy
- Simple financial management for everyday users
- Clean Capitec-style UI for fast budgeting

## Slide 2: What is BudgetBuddy?
- A browser-based budgeting app built with HTML, CSS, and JavaScript
- Tracks income, expenses, savings goals, wallet balance, and reports
- Includes login/register flow and offline-ready PWA support

## Slide 3: Why it was built
- Problem: People struggle to track spending and savings using manual notes
- Problem: Many budgeting apps are complex or require account setup
- Solution: BudgetBuddy gives a local, instant budgeting tool in the browser

## Slide 4: Features
- Register and login securely using local storage
- Add income and expense entries
- Track savings goals and wallet balance
- View reports and export transactions as PDF
- Ask the AI assistant for simple budget advice
- Dark mode, responsive layout, and PWA support

## Slide 5: How it works
- `index.html` is the app entry page
- `register.html` stores user credentials locally
- `login.html` validates credentials and opens the dashboard
- `dashboard.html` shows the financial summary and chart
- Shared logic lives in `Javascript.js`
- Data persists in the browser using `localStorage`

## Slide 6: Benefits
- Fast startup without installation
- Keeps all data locally on the device
- Easy to use for beginners and students
- Professional, bank-style interface for financial confidence

## Slide 7: Where and when to use it
- Open `index.html` in any modern browser
- Works offline after the first load thanks to service worker caching
- Ideal for daily spending checks, goal tracking, and short-term budgeting
